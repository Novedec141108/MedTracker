QUICK START GUIDE
=================

1. Open PowerShell in the medicine-tracker folder

2. Install Node packages:
   npm install

3. Set email credentials (optional, but required for reminders):
   $env:EMAIL_USER = "your-gmail@gmail.com"
   $env:EMAIL_PASS = "your-gmail-app-password"
   
   Get your app password here: https://support.google.com/accounts/answer/185833

4. Start the server:
   npm start
   
   Output should say:
   "Medicine Tracker Server running on http://localhost:3000"

5. Open in browser:
   http://localhost:3000/index.html
   Or simply open the index.html file directly

6. Add a user:
   - Name: Your name
   - Email: Your email (optional but recommended for reminders)
   
7. Add medicines to that user

8. Click any date in the calendar to mark medicines taken

9. Daily reminders: Sent automatically at 8 AM to all users with emails

10. Monthly report: Sent automatically at the start of each month


IMPORTANT NOTES:
- Data persists in localStorage (browser) and tracking-data.json (server)
- For production, replace nodemailer with your email service (SendGrid, AWS SES, etc.)
- Currently uses Gmail with app password; update server.js for other providers
